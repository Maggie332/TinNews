# TinNews
> An Android-based Tinder-like news app.

## Table of Contents
* [General Info](#general-information)
* [Technologies Used](#technologies-used)
* [Features](#features)
* [Screenshots](#screenshots)


## General Information
- This Android app was developed based on Google Component Architectural MVVM pattern and using Room for local database operations.


## Technologies Used
- MVVM pattern, Jetpack navigation, Retrofit and OkHttp, Gson, RecyclerView, Picasso library
- External resources: 
    -  NewsAPI.org for news feed 
- Database: local databse with Room


## Features
- Home page for news feed
- Like/dislike news
- Search news
- Saved news
- Get news details
